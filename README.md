Project Title

E-Commerce Website

Description

A fully functional e-commerce website that allows users to browse products, add items to the shopping cart, and view order summaries. The project demonstrates web development skills and responsive UI design.

Features

Product listing with images and descriptions

Add/remove items from shopping cart

View order summary with total price

Responsive design for different screen sizes

Tech Stack
HTML, CSS, JavaScript

Frontend-focused (can integrate backend later)
